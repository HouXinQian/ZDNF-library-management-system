# 项目说明
    基于JavaWeb的图书馆管理系统，有管理员和学生两端
    
## 功能
### 管理员：
    （1）可对用户的信息进行增删查改操作
    （2）能添加和删除用户
    （3）对图书、图书卡和图书类别的增删查改管理
    （4）管理用户的借还书的情况
### 学生：
    （1）借阅或归还图书
    （2）查找图书
    （3）查看图书卡

## 技术
    （1）以layui框架为前端
    （2）以Java为后端，采用MVC模式开发，使用Servlet架构
